# Sportswear9

Frontend app for the Sportswear9 project. The Vite app lives in `frontend/`, and the repo root provides convenience npm scripts so you can work from either location.

## Run From Repo Root

```powershell
npm run dev
```

## Install Frontend Dependencies

```powershell
npm run install:frontend
```

## Available Root Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`
